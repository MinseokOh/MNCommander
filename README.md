## System Requirements

 * Ubuntu 16.04 x64
 * Minimum Storage 10GB
 * Minimum Memory 2GB
 
## How To Use

```sh
git clone https://github.com/MinseokOh/MNCommander.git
cd MNCommander
bash MNCommander.sh
```
